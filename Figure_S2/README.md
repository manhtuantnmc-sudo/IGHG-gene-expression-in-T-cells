# README – RNA-seq Analysis Code (FigureS2)

## Overview
This repository contains the R code used to generate the analyses and figures corresponding to Figure 1 in the manuscript.

The data used in this study are publicly available from the Gene Expression Omnibus (GEO) database under the accession number:

GSE126030

Because the dataset is publicly available, the raw data are not included in this repository.
Please download the data directly from GEO and run the analysis scripts provided here.

---

## Data Source

Database: Gene Expression Omnibus (GEO)  
Accession number: GSE126030  

Study link:  
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE126030

The following files from the dataset were used in this analysis:

- Count matrix (.mtx)
- Column annotation file (.mtx.column)
- Gene annotation file (.mtx.row)
- Sample metadata

---

## Reproducibility

To reproduce the analysis:

1. Download the dataset from GEO (GSE126030)
2. Place the required files in the working directory
3. Run the provided R script for FigureS2 analysis

The script will reproduce all analyses and figures reported in FigureS2 of the manuscript.
